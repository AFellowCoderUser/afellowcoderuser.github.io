Very Basics to CSS

Here, I'll be explaining on how to make a CSS rulesets and other handy things.

A CSS ruleset will look like something like this:

h1 {
    font-weight: bold
}

Here, this ruleset here says that all <h1> tags will be in bold without the use of the <bold> tag. You shouldn't have any empty rulesets present in your CSS file. The last shorthand property (In the example code it's font-weight) may have no semicolon on the end. However, if you want to add more properties to the ruleset, first add a semicolon to the previous property and repeat on the others.

Like this:

h1 {
    font-weight: bold; /*Semicolon added because there is another property added below.*/
    color: white; /*Semicolon added. Is optional until another property is added.*/
}

Styles to Specific Tags

You can assign a specific style to either a HTML tag, ID, or Class. For example:

p {
    color: red;
}

This code says that all <p> tags with text between the tags will be colored red.

#red {
    color: red;
}

This code says that all elements that are given the id="red" attribute will color the text within it red. IDs are not to be used more than once.

.red {
    color: red;
}

This code says that all elements that are given the class="red" attribute will color the text within it red. This class can be assigned to more than one tag.

CSS without CSS files

HTML should already recognize CSS code, whether it's coming from an external source or within the HTML file. You can either use the <style></style> tags in the <head>, or use style="" within a HTML tag. For example:

<style>
    .center {
        text-align: center;
    }
</style>

This is a very similar way to how you would write CSS code in an actual CSS file, except this CSS code is only present in the HTML file you are currently editing right now. This is useful if you are planning for the CSS code to only be used in one HTML file. If you want a specific tag to have this style, without the use of IDs or Classes, you can use:

<p style="text-align:center;"></p>

Here, only that specific <p> tag will have its text aligned to the center. I recommend using this method of styling if you're planning not to use this multiple times, especially when it's long.
