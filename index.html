Getting Started

Now, this guide focuses on making your own personal, static website, so if you're looking for a place where you can sell your stuff or something similar to that, this guide is not for you.

Now that's out of the way, I will just introduce you to what web hosting service you'll be using for your personal, static website right off the bat.

Neocities!

Neocities is a free web hosting service for static pages. The free plan that it offers is pretty good depending on what website you are planning to host. The paid plan is also pretty good, only $5/mo and you're given more features like a custom domain!

Now, to get started on Neocities, create your account (of course). Once your account is created, your website is also created at the same time with all of the necessary files it needs.

We'll be focusing on these two files:

  • index.html
  • style.css

index.html is the main file of your website, anyone who types in the address of your website (like username.neocities.org) is directed to there. The file is also pre-written with html code so you're not stranded with a blank file. Same goes for the "style.css" file, the "index.html" file should already link the css file with .
