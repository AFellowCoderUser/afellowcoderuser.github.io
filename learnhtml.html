Learning HTML

HTML is the basic skeleton of a website. HTML is a text document that tells whatever browser you are using to display the information written on the file. A website simply cannot exist without HTML files because there is no file to be displayed in the browser.

This guide will take you through the basic steps of HTML.

Getting Started on HTML

You may create another HTML file on NeoCities, or in general just use any HTML file. I recommend creating a new blank HTML file.

With your new HTML file created, your new HTML file on NeoCities should have some code already pre-written. If your new HTML file isn't on NeoCities, here's what should be on your HTML file:

                        
<!DOCTYPE html>
<html>
    <head>
        <meta charset="UTF-8">
        <title>Your Page Title</title>
        <link rel="stylesheet" type="text/css" href="/style.css" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
    </head>
    <body>
        <p>This is a paragraph</p>
    </body>
</html>

Copy and paste the code into your HTML file. This is the base of a website page. Now, lets walk through what all of that code means.

    <!DOCTYPE html> tells the browser that this is a HTML file. While webpages can work without that declaration, without it, it'll cause some weird behavior.

    <html> is used to contain the <head> and <body>. Sometimes, it may also say <html lang="en">, this states that the page is in english.

    <head> is what contains meta information of your site. <meta charset="UTF-8"> is used to specify that the webpage uses the UTF-8 character set. This is needed to display symbols. The second meta tag is used to make sure that the webpage (sometimes) looks good on mobile. <title>Your Page Title</title> is the page's title displayed on the tab. Finally, the <link> is used to link CSS files to the page, it's also used to link images to become the website's favicon.

    <body> is where all of the webpage's content is held in. Most of the website's code will be placed there.

    Finally, it is closed off with </html>. Nothing should be outside either the <head> and <body> tags.
