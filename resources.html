Resources

There are many web development resources out there that can possibly explain more things much better and in detail.

MDN Documentation - Excellant for referencing HTML/CSS/JS. Useful if you don't know what "this" means.

W3Schools - Has many coding tutorials and guides in various different programming languages.

Sadgrl's webmaster guides - A guide that explains some things mentioned in this website more in detail + HTML/CSS code snippets.

Deploy to NeoCities - Although NeoCities has a feature where you can edit your site with a external code editor of your choice, you'll need a supporter plan to do such a thing. This is when Deploy to NeoCities comes in.

Deploy to NeoCities utilizes your website's API key to create, edit, and delete files on your website's storage. You can host all of your website's files on a Github repository and if you commit and push changes to your repository, a bot will gather all of the files and deploy those files in your website's storages via your website's API. This is much better than just going to the website then making changes to each file and saving. With Deploy to NeoCities, you can commit and push the changes all at once.

This site's repository - You can use all of the code here as a reference on making your own website.

File Garden - Previously mentioned in the Intro page, this is a file hosting website. Useful if you want to free up storage space on your website's storage.

There's so much things to learn, but you'll eventually master it all! Your website WILL eventually evolve to the style that you wanted! Decorate it, make it interactive, it's your website!
